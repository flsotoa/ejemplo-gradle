# Ejemplo-gradle

Getting Started

Comandos para ejecutar en Windows:

•	Compile, test, jar:
  gradle build

•	Run:
  gradle bootRun 

•	Test:

  curl -X GET 'http://localhost:8081/rest/mscovid/test?msg=testing'
  
  O para revisar estado mundial ejecutar:
  curl -X GET 'http://localhost:8081/rest/mscovid/estadoMundial'
